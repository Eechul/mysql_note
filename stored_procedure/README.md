## 저장프로시저란?

일련의 쿼리를 마치 하나의 함수처럼 실행하기 위한 쿼리문들의 집합

## 장점

- 캐시에 있는 실행계획을 가져오기 때문에 속도가 빠르다
- 유지보수 및 재활용 측면에서 좋다
- 클라이언트에서 서버로 긴 쿼리문을 전송하지 않아도 되기 때문에 네트워크 부하가 적다

## 단점

- 서비스를 확장할때 서버수를 늘리는 대신 DB수를 늘려야 하는데, 상대적으로 DB수를 늘리는게 어렵다.
- 버전관리가 힘들다
